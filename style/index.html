Getting Start（初步认识 CSS ）

CSS（层叠样式表）是一种用于描述 HTML 文档外观的样式表语言。它允许开发者控制网页的布局、颜色、字体等视觉效果。

CSS 的基本语法包括 选择器 和 声明块 。选择器用于选择要应用样式的 HTML 元素，而声明块则包含一组属性和对应的值。

例如，以下 CSS 规则将所有段落的文本颜色设置为蓝色：

p {
    color: blue;
}

在这个例子中， p 是选择器，表示所有的段落元素。 color 是属性， blue 是值。

CSS 还支持多种选择器，如类选择器、ID 选择器和伪类选择器等，可以帮助开发者更精确地选择要应用样式的元素。

通过使用 CSS，开发者可以创建美观、易于使用和响应式的网站。

在本节中，我们将介绍 CSS 的基本概念和语法，并提供一些简单的示例来帮助您入门。
