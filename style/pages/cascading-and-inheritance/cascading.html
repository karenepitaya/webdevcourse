Cascading and Inheritance（层叠与继承）

CSS 的层叠性允许多个样式规则应用于同一元素。浏览器根据特定的优先级规则决定使用哪个样式。

例如：

h1 { color: red; } /* 规则 1 */
h1 { color: blue; } /* 规则 2 */

在这个例子中，h1 元素的颜色将是蓝色，因为规则 2 在规则 1 之后定义。

CSS 的继承性允许某些属性从父元素传递到子元素。例如，字体和颜色属性通常会被继承。
