Selectors（ CSS 选择器）

CSS 选择器用于选择要应用样式的 HTML 元素。常见的选择器包括元素选择器、类选择器和 ID 选择器。

例如，以下 CSS 规则将所有段落的文本颜色设置为蓝色：

p { color: blue; }

在这个例子中，p 是元素选择器，表示所有的段落元素。

类选择器以点号（.）开头，用于选择具有特定类的元素。例如：

选择器的种类

  • 元素选择器（Element Selector）: 
    h1 {
}
  • 类选择器（Class Selector）: 
    .box {
}
  • ID 选择器（ID Selector）: 
    #unique {
}
  • 属性选择器（Attribute Selector）: 
    a[title] {
}
a[href="https://example.com"] {
}
  • 伪类选择器（Pseudo-class Selector）: 
    a:hover {
}
  • 伪元素选择器（Pseudo-element Selector）: 
    p::first-line {
}

属性选择器与字符串匹配运算符

<h1>Attribute presence and value selectors</h1>
<ul>
  <li>Item 1</li>
  <li class="a">Item 2</li>
  <li class="a b">Item 3</li>
  <li class="ab">Item 4</li>
  <li class="a-b">Item 5</li>
</ul>
body {
  font-family: sans-serif;
}
/* 使用 li[class]，我们就能匹配任何有 class 属性的选择器。这匹配了除了第一项以外的所有项。 */
li[class] {
  font-size: 120%;
}

/* li[class="a"]匹配仅有一个 a 类的选择器，它选中了第二项。 */
li[class="a"] {
  background-color: yellow;
}

/* li[class~="a"]会匹配一个a类，不过也可以匹配一列用空格分开、包含a类的值，它选中了第二和第三项。 */
li[class~="a"] {
  color: red;
}

/* li[class|="a"]会匹配一个a类，不过也可以匹配以 a 开头后接短杠"-"的值，它选中了第二和第五项。 */
li[class|="a"] {
  font-style: italic;
}

<h1>Attribute substring matching selectors</h1>
<ul>
  <li class="a">Item 1</li>
  <li class="ab">Item 2</li>
  <li class="bca">Item 3</li>
  <li class="bcabc">Item 4</li>
</ul>
body {
  font-family: sans-serif;
}

/* li[class^="a"]匹配了任何值开头为a的属性，于是匹配了前两项。 */
li[class^="a"] {
  font-size: 120%;
}

/* li[class$="a"]匹配了任何值结尾为a的属性，于是匹配了第一和第三项。 */
li[class$="a"] {
  background-color: yellow;
}

/* li[class*="a"]匹配了任何值的字符串中出现了a的属性，于是匹配了所有项。 */
li[class*="a"] {
  color: red;
}

伪类选择器与伪元素选择器

伪类是选择器的一种，它用于选择处于特定状态的元素，比如当它们是这一类型的第一个元素时， 或者是当鼠标指针悬浮在元素上面的时候。它们表现得会像是你向你的文档的某个部分应用了一个类一样， 帮你在你的标记文本中减少多余的类，让你的代码更灵活、更易于维护。


伪元素也是选择器的一种，它用于选择元素的特定部分，比如第一个字母或者第一个行。 它们表现得会像是你向你的文档的某个部分应用了一个类一样，帮你在你的标记文本中减少多余的类， 让你的代码更灵活、更易于维护。


伪类和伪元素选择器的语法是一样的，都是在选择器后面加冒号（:），后面跟上伪类或伪元素的名称。

伪类选择器的名称前面有一个冒号（:），而伪元素选择器的名称前面有两个冒号（::）。

关系选择器

关系选择器用于选择与其他元素有特定关系的元素。它们在其他选择器之间和其他选择器与文档内容的位置之间建立了一种有用的关系的缘故。

  • 后代选择器（空格）: 匹配第一个选择器里的元素的后代元素。无论是子代还是什么孙代还是曾孙代。 
    body article p
  • 子代选择器（>）: 只匹配第一个选择器里的元素的子代元素。注意与上面那个区分 
    article > p
  • 相邻兄弟选择器（+）: 用来选中恰好处于另一个在继承关系上同级的元素旁边的物件 
    p + img
  • 一般兄弟选择器（~）: 用来选中处于在继承关系上同级的元素 
    p ~ img
