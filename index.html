MDN 的全称是 Mozilla Developer Network，中文一般称为 Mozilla 开发者网络。
不过需要注意：虽然“MDN”最初代表 Mozilla Developer Network，但现在它的官方名称是 MDN Web Docs，这是因为：

  • MDN Web Docs 已不仅限于 Mozilla 自家技术（如 Firefox 或 Gecko），而是覆盖了整个 Web 技术生态（如 HTML、CSS、JavaScript、API 规范等）。
  • 它是由 Mozilla 主导维护，但有许多开发者和公司（如 Google、Microsoft、Apple）也会贡献内容。
